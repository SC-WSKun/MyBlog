# ES6简介

> ECMAScript是为JavaScript制定的标准，前者是后者的标准，后者是前者的一种实现

`ES6`泛指`ES2015`以及`ES2015`后的所有版本

## Babel转码器

> Babel是一个广泛使用的ES6转码器，可以让ES6的代码可以在老版本的浏览器执行，而不用担心现有环境是否支持
